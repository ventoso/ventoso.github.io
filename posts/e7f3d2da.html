天翼云盘

天翼云盘，国内市场中为数不多的良心云盘之一，电信号老用户有10tb或2tb固定云存储空间，新号或者其他运营商的账号也有30g云存储空间，存个小东西什么的也够用，但是良心的电信也总推出限时会员的活动，有4tb个人云存储空间和2tb家庭云空间。到期不删资源还是很不错的。

sharelist

项目传送门
sharelist是reruin大佬node.js开发的云盘目录程序，前端页面简洁美观，同时支持多种云盘不同方式的挂载（cookie/api/分享链接）

GoogleDrive
OneDrive（含世纪互联）
天翼云盘（支持账号密码挂载）
本地文件
GitHub
蓝奏云
h5ai
WebDAV
虚拟目录

功能强大，支持页面自适应与媒体影音播放，图片在线查看，流量中转等功能。且作者也在不断地进行着更新与维护。

程序部署

需要在vps/nas下部署

docker部署

部署docker

以root或sudo权限登录centos。
更新yum包

sudo yum update

逐行输入并执行一键安装脚本

curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh

启动docker进程

sudo systemctl start docker

到此docker部署完毕，后续可以进行更换镜像地址，查看docker版本等操作。

docker部署sharelist

在docker环境中输入以下代码并执行，等待操作完毕后，通过服务器ip地址:33001来访问sharelist并进行下一步操作。

docker run -d --name=sharelist --restart=always -p 33001:33001 -v /root/sharelist:/app/cache reruin/sharelist

ps：端口默认33001 路径默认/root/sharelist自行修改。

至此，sharelist使用docker部署完成。

shell安装

前往项目传送门打包下载源码

然后将下载后的zip文件上传至服务器网站目录，并解压文件。

进入ssh

cd /文件目录

输入指令进行安装

bash install.sh

等待安装即可，安装完成通过服务器ip地址:33001来访问sharelist并进行下一步操作。

远程安装/netinstall

wget --no-check-certificate -qO-  https://raw.githubusercontent.com/reruin/sharelist/master/netinstall.sh | bash

更新 / Update

cd /程序目录
bash update.sh

天翼云网盘挂载（cookie/api）

访问ip:端口进入初始化界面，自定义新口令（密码），网站标题等，点击保存，看是否能正常挂载filesystem

下拉点击管理，进入管理界面，输入刚才新建的密码来访问后台

添加虚拟目录。

这里只说一下天翼云盘挂载的两种方法，其他网盘类比一下，方法大同小异。

cookie(账号密码挂载)

此类方法适用于无api的用户来挂载，访问速度不是特别快

添加虚拟目录选择

名称自定义，目录只填/，然后重新进入网站，进入所添加的虚拟目录，根据提示填入账号密码即可挂载。

api挂载

添加目录选择

名称自定义，目录仍只填入/

如果你跟我一样使用的是github开源api，这时候先不要着急返回网站，这时候需要你在你自己的天翼云盘账号我的应用路径下新建一个safebox的文件夹

因为开源api挂载的是全部文件>我的应用>safebox目录内的内容的

然后返回网站，进入添加的虚拟目录，根据提示输入key和secret，然后跳转至天翼云，允许授权即可

配置域名

这里说一下宝塔面板配置域名的方法

新建一个网站

可以在设置反 代之前配置一下ssl

点击设置
来配置反代

目标url中只填如程序网址即可，且为http://ip:端口格式，末尾不必填/

提交即可，接着就可以用域名来访问网站了。
